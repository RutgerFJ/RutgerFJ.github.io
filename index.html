BISA

HTML - CSS - JS - Python - Django - SQLite - HTMX

Stravis

My first Django application. After using Strava to track my sports activities for about 3 years, I longed for a better way to visualize my activities. Hence the name "Stravis". The application allows users to connect to their Strava account, download their activities and creates multiple customizable plots.

HTML - CSS - JS - Python - Django - SQLite - D3 - Bokeh

MasterBlast

This application was part of my Minor Application Development during my study. Our group of 4 students worked with another group of students as a client. The client tasked us with creating a wrapper for NCBI's BLAST (a bio-informatics tool). The application allowed users to store their BLAST queries and results, aswell as share them with other users.

HTML - CSS - JS - Python - Django - SQLite

Reedsnake

A Snakemake pipeline for the analysis of DNA sequencing data. Wrote custom scripts for the quality control and trimming of the reads and used existing tools to perform both a reference-based and a de novo assembly.

Python - Snakemake - Bash

COGselect

The application that made me fall in love with web development! The application is very simple, allowing the user to connect to a PostgreSQL database and download clusters of orthologous genes (COGs). The application visualizes these clusters and allows filtering, so the user can select clusters that they deem interesting.

HTML - CSS - Python - Flask - PostgreSQL
